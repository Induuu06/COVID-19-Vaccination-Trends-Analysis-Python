# COVID-19 Vaccination Trends Analysis — Python

**Tool:** Python | **Course:** BUSN5101 Programming for Business | **University:** University of Western Australia | **Group 33**

## What this project does

Analyses a global COVID-19 vaccination dataset to identify trends, compare countries, and visualise vaccination progress from 2020 to 2022.

## Libraries used
- `pandas` — data cleaning and analysis
- `matplotlib` — line chart visualisations

## Key tasks
1. Top and bottom 5 countries by full vaccination rate in 2021 and 2022
2. Date with the highest single-day vaccinations worldwide (June 27, 2021 — 44M doses)
3. Top and bottom 5 countries by total vaccinations on June 27, 2021
4. Line chart visualisation of vaccination trends (Oct 2020 – Oct 2022)

## Files
- `Final_code.py` — main Python script
- `data/cleanedDF3.csv` — cleaned dataset
- `report/BUSN5101_Group33_Report.pdf` — full written report
